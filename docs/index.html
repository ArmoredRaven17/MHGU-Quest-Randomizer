      How to use

      1. Pick a quest type & level. Choose a Quest Type (Village, Hub, G-Rank, Special Permits, Events, Arena, or ALL), then set a From / Up to Level range.

      2. Narrow the pool with the sidebar filters:

      • Quest Filters — uncheck categories to exclude them (all are on by default except Arena). Hypers, Captures, and Multi Monster sit under Large Monsters and are also excluded when it is off. The S. Permit filters (One Faint, On-Site Items) restrict those Special Permit quests. Quest Type Filters let you disable individual levels (e.g. turn off all G-Rank levels globally).
      • Monsters / Weapons / Styles — uncheck anything to exclude it from the roll.
      • Hunter Arts — uncheck arts to exclude them, by weapon, by named art, or by individual level (I / II / III).
      • Restrictions — add weapon + style combos that will never be rolled together (e.g. Bow + Aerial).
      • Challenges — add your own conditions (e.g. "No Armor"), each with a chance %, that roll alongside the quest. Set how many can apply per quest, and toggle each one on/off. Names and chances are editable inline.
      • Prowler — enable "Prowler?" to allow Palico loadouts; "Prowler Quests?" to include Prowler-only quests. Biases apply when a Prowler is rolled.

      3. Hit Randomize. You get a quest, a weapon, style, and Hunter Arts (plus any rolled challenges). Arena quests show which weapon Set to bring; Prowler quests give a bias instead.

      Quest badges: Hunt Small Monsters Egg Delivery Gathering Special Permit Capture Hyper Prowler Arena Event

      ⚙ Theme — change the color theme or pick a Guild Card background image.

      Export / Import — save all your filters and challenges to a file, or load them back later (handy for sharing a setup or moving between browsers).

      Filters, challenges, theme, and background are saved between visits. Use Reset Filters to restore filter defaults (your challenges are kept).

      Reset Filters?

      All monsters, weapons, styles, Hunter Arts, and restrictions will be restored to their defaults.
