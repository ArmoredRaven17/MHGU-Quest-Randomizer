    How to use

    1. Pick a quest type & level. Choose a Quest Type, then the From / Up to Level range. (Arena uses a single All / Normal / Challenge picker instead of a range.)

    2. Narrow the pool with the filters on the left:

    • Quest Filters — large-monster quests are always included; check Hypers, Egg Delivery, Gathering, or Small Monsters to add those categories.
    • Monsters / Weapons / Styles — uncheck anything to exclude it from the roll.
    • Hunter Arts — uncheck arts to exclude them, by weapon, by named art, or by individual level.
    • Prowler — turn on "Prowler?" to allow Palico loadouts, "Prowler Quests?" to include Prowler-only quests; the biases apply when a Prowler is rolled.

    3. Hit Randomize. You get a quest plus a weapon, style, and Hunter Arts. Arena quests show which preset Set to bring; Prowler quests give a bias instead; a yellow Hyper badge marks Hyper quests.

    Your filters and color theme are saved between visits. Use Reset Filters to start over.
